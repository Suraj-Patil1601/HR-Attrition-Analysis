# HR-Attrition-Analysis
👨‍💼 HR Attrition Analysis – Understanding Factors Influencing Employee Turnover
📌 Project Overview
This project analyzes employee attrition data to identify the key factors influencing employee turnover. Using Exploratory Data Analysis (EDA), the project uncovers patterns related to employee demographics, job satisfaction, compensation, overtime, and work experience to generate actionable business insights.

🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Plotly

📊 Analysis Performed
Data Cleaning and Wrangling
Missing Value Analysis
Attrition Distribution Analysis
Age and Income Analysis
Department-wise Attrition Analysis
Job Satisfaction Analysis

📂 Dataset Information
Total Records: 1,470
Total Features: 35
Target Variable: Attrition (Yes/No)
